# 📝 API To-Do List com Node.js, Express e Prisma

Este é um projeto de uma API REST para gerenciamento de tarefas (to-do list), desenvolvido com Node.js, Express e Prisma ORM. Ideal para quem está aprendendo a construir APIs com banco de dados de verdade.

---

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- Prisma ORM
- SQLite (banco de dados)
- Nodemon (para desenvolvimento)
- JavaScript ESModules

---

## 📚 Funcionalidades

A API permite:

- ✅ Criar uma nova tarefa
- 📋 Listar todas as tarefas
- ✏️ Atualizar uma tarefa
- ❌ Deletar uma tarefa
- ✔️ Validações via middlewares
- 💾 Dados persistidos em banco de dados SQLite
